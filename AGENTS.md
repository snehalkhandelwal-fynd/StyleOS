## Core Product Direction

This is not a generic e-commerce app.

Do not default to:
- product grids
- category-heavy browsing
- marketplace-style banners
- discount-first layouts

The app should feel like a personal fashion decision system.

The core experience is:

Discover → Try on → Evaluate → Style → Save / Buy / Reuse with closet

## Primary UX Goals

Every screen should help users:
- decide what to wear or buy
- see how outfits look on them
- understand how to style items
- reuse their existing wardrobe
- reduce uncertainty before buying

## Business Priorities

Prioritize flows that increase:
- first try-on
- closet upload
- saved looks
- style-profile completion
- return visits
- purchase confidence
- wardrobe-based recommendations

## Design Rules

Use large visuals, clean spacing, and outfit-first cards.

Primary CTAs should be clear and action-led:
- Try on me
- Add first item
- Ask Mira
- Save look
- Style this

Avoid clutter, fake personalization, and unnecessary marketplace patterns.

## Personalization Rule

Do not use labels like “For you”, “Your style”, “91% match”, or “Styled for you” unless the app has enough user signals from onboarding, saved looks, closet upload, search behavior, try-on history, or swipe quiz answers.

For new users, use neutral labels:
- Popular
- Trending
- Try now
- Work-ready
- New this week

## Closet Rule

Closet upload is strategically important but should not feel like work.

Use:
- Add one item
- Add 3 pieces you love
- Build looks from what you own
- Style something you already have

Avoid:
- Upload your wardrobe
- Complete your closet
- Add 10 items to unlock
- Upload 10 items to unlock your colour analysis
- Upload 10 items to unlock your color analysis

## Mira Rule

The AI stylist is named Mira.

Mira should help users with:
- how to style an item
- what to wear for an occasion
- what is trending
- what goes with a closet item
- budget-based looks
- outfit decisions
- wardrobe reuse

Mira should always connect advice back to actions:
- view looks
- try on
- upload closet item
- save
- shop
- compare options
